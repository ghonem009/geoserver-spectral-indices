# Spectral Index API

**Training Project:** Calculate and serve spectral indices (NDVI, NDWI, NDBI, MNDWI) with Python, FastAPI, and GDAL for GeoServer.

## Features

- Compute NDVI, NDWI, NDBI, MNDWI from input raster bands.
- Serve outputs via FastAPI endpoints.
- Save results as GeoTIFF and optional SLD (style) for GeoServer.
- Easy upload to GeoServer using REST API.
